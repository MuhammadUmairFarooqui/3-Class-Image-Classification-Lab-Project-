## Project Description

This project classifies images into 3 classes: Airplane, Car, Bird using three AI models:

1. Machine Learning (Random Forest)
2. Deep Learning (CNN)
3. Agentic AI (GROQ Llama 3.3)

## Dataset

CIFAR-10 (3 classes only)
Link: https://www.cs.toronto.edu/~kriz/cifar.html
